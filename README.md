# Guess the Number – Python

This project is a simple number guessing game written in Python.

## Description

The program randomly generates a number between **1 and 100**.
The player must guess the correct number by entering a value from the keyboard.

After each attempt, the program informs the player whether the guess was correct and shows the correct number.

## Features

* random number generation
* user input from keyboard
* attempt counter
* simple game loop

## Technologies

* Python
* random module

## How to run

1. Install Python.
2. Run the script:

```bash
python guess_number.py
```

3. Enter a number between **1 and 100** and try to guess the correct value.

```text
Podaj liczbe: 50
Nie trafiles
Poprawna odpowiedz to 73
```
